# Aplikasi Gudang Logistik

Aplikasi Gudang Logistik adalah aplikasi desktop yang dibuat menggunakan **Java Swing** dan **MySQL** sebagai tugas mata kuliah **Pemrograman 2**.

## Fitur

- Menambah data barang
- Menampilkan data barang
- Mengubah data barang
- Menghapus data barang
- Mencari data barang

## Teknologi

- Java
- Java Swing
- JDBC
- MySQL
- Apache NetBeans 21

## Struktur Project

```
aplikasiGudangLogistik
│
├── src/
├── nbproject/
├── database/
│   └── gudang_logistik.sql
├── build.xml
├── manifest.mf
└── README.md
```

## Cara Menjalankan

1. Import database `gudang_logistik.sql` ke MySQL menggunakan phpMyAdmin.
2. Buka project menggunakan Apache NetBeans.
3. Pastikan konfigurasi koneksi database sudah sesuai.
4. Jalankan aplikasi.

## Pengembang

**Jajang Solihin**

Mata Kuliah: **Pemrograman 2**
